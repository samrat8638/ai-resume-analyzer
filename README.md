🚀 AI Resume Analyzer with Interview Preparation
📌 Overview

AI Resume Analyzer is a full-stack web application that analyzes resumes and generates personalized interview questions using AI. Built with Django and integrated with Google Generative AI API, this project helps users improve their interview readiness.

✨ Features
📄 Upload Resume (PDF support)
🤖 AI-based Resume Analysis
🎯 Skill Extraction from Resume
💡 Personalized Interview Questions
🔐 User Authentication System
⚡ Dynamic UI with JavaScript (no page reload)
🛠️ Tech Stack
Backend: Python, Django
Frontend: HTML, CSS, JavaScript
Database: SQLite
AI Integration: Google Generative AI API
Tools: Git, GitHub
📂 Project Structure
ai_resume/
│── analyzer/

🚀 How to Run Locally
1️⃣ Clone the repository
git clone https://github.com/samrat8638/ai-resume-analyzer.git
cd ai-resume-analyzer
2️⃣ Create virtual environment
python -m venv venv
3️⃣ Activate environment
venv\Scripts\activate
4️⃣ Install dependencies
pip install -r requirements.txt
5️⃣ Run server
python manage.py runserver
